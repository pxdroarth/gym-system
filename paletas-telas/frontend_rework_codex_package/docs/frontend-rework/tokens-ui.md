# Tokens visuais sugeridos

## Cores
- primary-50: azul muito claro
- primary-100: azul claro de apoio
- primary-500: azul institucional principal
- primary-gradient: usar referência de `paleta/gradiente-azul-referencia.jpg`
- success: verde suave
- warning: laranja suave
- danger: vermelho suave
- neutral-bg: branco / branco-azulado
- neutral-border: cinza muito leve
- neutral-text: azul-acinzentado escuro

## Radius
- cards: médio para suave
- inputs: suave
- modais: suave/premium
- pills e badges: arredondado

## Sombras
- baixa intensidade
- apenas separação sutil de plano
- evitar profundidade agressiva

## Espaçamento
- usar respiro generoso
- evitar blocos colados
- priorizar leitura e hierarquia

## Tipografia
- aparência semelhante a Inter / sistema moderna e limpa
- títulos fortes
- subtítulos claros
- labels discretas
- dados densos com legibilidade alta
